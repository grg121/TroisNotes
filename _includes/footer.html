LE SECRET DU REFLET

"Un reflet sur les yeux
Révélait les couleurs
Imprégnés de douceur

Il m'a dit
Qu'il brille pour
Notres rires

Il m'a dit
Qu'il veut nous
Découvrir

Il peuplait...
Le bleu, soyeux,
Tout le temps, Tous les lieux,
La mer, la fraîche
Tous les faits, Tous les voeux

Il m'a dit
"Trois notes sont
Mélodie"

Il m'a dit
"Viens, tu peux
soit tranquille.

J'ai caché un secret.
Tout le miel, Tout le ciel,
Dans les deux, Sur les yeux."

C.
